# 🅱️Bootstrap-Day Project

A modern and responsive front-end project built using **Bootstrap** and custom **CSS**.  
This project focuses on clean design, smooth layout, and a fully responsive user experience across all devices.

## 📌 Features

- Fully responsive design (mobile, tablet, desktop)
- Built with Bootstrap framework
- Custom CSS styling for unique appearance
- Clean and modern UI
- Smooth scrolling navigation
- Structured and readable code

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5

## 💡 About the Project

This project was developed as a practice and showcase of front-end development skills using Bootstrap along with custom styling.  
The goal was to create a responsive and visually appealing layout while keeping the code clean and maintainable.
